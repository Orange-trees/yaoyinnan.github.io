虚拟环境

使用虚拟环境的目的：当出现某些功能依赖不同的时候，同一个python不应该承载太多的依赖包，创建相应功能的虚拟环境，在不同虚拟环境下安装依赖包，独立开来。

在相应的虚拟环境下执行：

virtualenvwrapper

设置环境变量 WORKON_HOME：D:\env

D:\env是虚拟环境存放的地址

virtualenvwrapper基本使用

1.创建虚拟环境

　　这样会在WORKON_HOME变量指定的目录下新建名为venv的虚拟环境。

　　若想指定python版本，可通过”–python”指定python解释器

2. 基本命令
