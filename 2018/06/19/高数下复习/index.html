多元函数极值、条件极值和最值

  1. 非条件极值：
    1. 求驻点 偏导x/y = 0 （x0， y0）
    2. 求驻点处（x0， y0）的 三个二阶偏导 A = xx， B = xy，c = yy
    3. 结论：
      1. AC - B^2 > 0 有极值 A \< 0 极大值 A > 0 极小值
      2. AC - B^2 < 0 无极值
      3. AC - B^2 = 0 不确定
  2. 条件极值：拉格朗日乘数法
  3. 最值：
    1. 求驻点
    2. 求边界点上的可能极值（最值）点（条件极值） 如：x^2 + y^2 <= 1 ==》 x^2 + y^2 = 1
    3. 比较上述点的函数值的大小
